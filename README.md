### JSDelivr安装地址

- 初始安装

```
curl -fsSL https://cdn.jsdelivr.net/gh/Xiaolong1811/mdserver-web@latest/scripts/install.sh | bash
```

- 直接更新

```
curl -fsSL https://cdn.jsdelivr.net/gh/Xiaolong1811/mdserver-web@latest/scripts/update.sh | bash
```

- 卸载脚本

```
wget -O uninstall.sh https://cdn.jsdelivr.net/gh/Xiaolong1811/mdserver-web@latest/scripts/uninstall.sh && bash uninstall.sh
```
